# 西安财经大学 iCalendar 校历

> 灵感：[武汉大学 iCalendar 校历](https://github.com/ExerciseBook/whu-calendar)

## 使用（开发者）
1. 拉取代码仓库
2. 运行 `yarn install`（请先确保已安装 NodeJS 和 Yarn）
3. 运行 `yarn run run`
4. 在根目录获得 `.ics` 档案文件

## 使用（普通用户）
请遵照 [该网站](https://calendar.xctra.cn) 的指引导入 iCalendar 档案
